# SciFig Scientific Figure Skill

The **SciFig Scientific Figure Skill** is an open-source skill for [Codex, Claude Code, and AI agents](https://scifig.ai/?ref=readthedocs "SciFig — AI scientific figure generator") that turns research ideas, paper methods, and grant roadmaps into **publication-ready scientific figures**. It is powered by [SciFig](https://scifig.ai/?ref=readthedocs "SciFig website") — an AI scientific illustrator built for scientific accuracy and editability.

## What it does

Describe a mechanism, paste a method section, or hand it a sketch, and the skill produces a clean, accurate, journal-ready figure where **every text label stays editable**. Unlike general-purpose image generators that hallucinate anatomy and scramble labels, SciFig is tuned for **scientific accuracy**, so the figure is one you can defend in peer review.

## Input modes

The skill maps to SciFig's seven tools:

- [Text to Figure](https://scifig.ai/app/text-to-figure?ref=readthedocs "SciFig Text to Figure") — describe a mechanism or workflow in plain language
- [Sketch to Figure](https://scifig.ai/app/sketch-to-figure?ref=readthedocs "SciFig Sketch to Figure") — hand-drawn doodle to polished figure
- [Photo to Figure](https://scifig.ai/app/photo-to-figure?ref=readthedocs "SciFig Photo to Figure") — lab or microscopy photo to clean schematic
- [Reference to Figure](https://scifig.ai/app/reference-to-figure?ref=readthedocs "SciFig Reference to Figure") — an existing figure as a reference for your own variant
- [PDF to Figure](https://scifig.ai/app/pdf-to-figure?ref=readthedocs "SciFig PDF to Figure") — a paper's method/results section to a visual figure
- [Figure Enhancer](https://scifig.ai/app/figure-enhancer?ref=readthedocs "SciFig Figure Enhancer") — multimodal refine, restyle, upgrade
- [Vector Canvas](https://scifig.ai/app/vector-canvas?ref=readthedocs "SciFig Vector Canvas") — convert any figure into a fully editable layered SVG

## Built for publication

- **Every label editable** after generation — no re-prompting
- **Editable PPTX** export with text layers intact
- **Layered SVG** vectors — Illustrator, Inkscape, PowerPoint compatible
- **8K PNG / JPG** with AI super-resolution for journal print sizes
